# navigate

✅ **Purpose**

Loads a specific URL in the browser.

## 🧩 Syntax

```typescript
{
  url: string
}
```

## 💬 Agent Example

```
"Navigate to https://example.com."
```

## 🛠️ Translated Primitive

```typescript
{
  type: "navigate",
  params: {
    url: "https://example.com"
  }
}
```
