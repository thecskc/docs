## How It Works

Each primitive action has:

- **Purpose**: What the action does
- **Syntax**: The structured format for direct editing
- **Agent Example**: How to describe it in natural language
- **Translated Primitive**: How Jina converts your natural language to structured syntax

<Tip>
  You can switch between Agent Mode and Primitive Mode at any time while editing test steps.
</Tip>
